# Nimra Manzoor - Portfolio Website

A personal portfolio website showcasing my projects and skills as a web developer.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Interactive project cards
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Or serve it using a local server for better experience

## Deployment

This is a static website that can be deployed to GitHub Pages, Netlify, or any static hosting service.
